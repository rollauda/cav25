# Introduction : Qu’est-ce que le cinéma ? 

:::warning[Exercice introductif]
**Qu'est-ce que le cinéma, selon vous ?   
Cherchez différentes définitions possibles.**

<u>[Aide] Posez-vous les questions suivantes :</u>
- A quel genre (domaine) appartient le cinéma ?
- Quel est son but, sa fonction ?
- Qu'est-ce qui le distingue et le rapproche d'autres disciplines proches ?
- De quoi est fait le cinéma, que "fabrique" ou crée-t-il ?
:::
